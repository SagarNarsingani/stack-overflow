## Run Locally

Clone the project

```bash
  git clone https://github.com/SagarNarsingani/stack-overflow.git
```

Go to the project directory

```bash
  cd stack-overflow
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

**Make sure that backend server is also running and is connected to database successfully**
